# OG Image Creation Guide

## Quick Instructions

Since image generation is currently unavailable, here's how to create your OG image:

### Option 1: Use Canva (Easiest)
1. Go to https://www.canva.com
2. Create new design: 1200 x 630 px
3. Use these colors:
   - Background: #0b0f19 (midnight blue)
   - Gradient: #8b5cf6 (purple) to #3b82f6 (blue)
4. Add text:
   - Main: "Muhammad.my Blog"
   - Subtitle: "Built with Antigravity"
5. Export as PNG
6. Save as `public/og-image.png`

### Option 2: Use Figma
1. Create 1200x630px frame
2. Apply dark background (#0b0f19)
3. Add gradient overlay (purple to blue)
4. Add text with Inter font
5. Export as PNG to `public/og-image.png`

### Option 3: Use Online Tool
- https://www.opengraph.xyz/
- https://www.bannerbear.com/tools/open-graph-image-generator/

### Design Guidelines
- **Background**: Dark midnight blue (#0b0f19)
- **Accent**: Purple-blue gradient
- **Font**: Inter or similar modern sans-serif
- **Style**: Minimalist, clean, tech-forward
- **Elements**: Abstract shapes, glassmorphism effects

## After Creating

1. Save the image as `public/og-image.png`
2. The SEO component is already configured to use it
3. Test with: https://www.opengraph.xyz/url/https://muhammad.my
